My Grid System

This is a responsive grid system made by me based on Bootstrap's Grid system. It has 12 columns that scales up or down based on device or viewport size. There are four predefined classes for PC(Width resolution > 1200px), Mini-Laptop/Netbook(Width resolution > 992px), Tablets(Width resolution > 768px), Mobile phones(Width resolution > 480px) and are as follows:

  • column-pc
  • column-lp
  • column-tb
  • column-mb

Basic html structure for using this grid system:

<div class="container">
<div class="row">
<div class="column-pc column-lp column-tb column-mb">
Your content goes here...
</div>
</div>
</div>

Also don't forget to add the link to the grid.css and Mediaquery meta tag in your head tag to get it working.

<meta name="viewport" content="width=device-width, initial-scale=1">

Note:
These classes should be postfixed with the appropriate values ranging from 1 to 12.

For example:
If we use the class column-pc12, then the content will take the whole row as a single column.
